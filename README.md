Getting Started
---------------

```sh
# clone it
git clone git@github.com/cdiezmoran/boilerplate-express-babel7.git
cd boilerplate-express-babel7

# Make it your own
rm -rf .git && git init && npm init

# Install dependencies
npm install

# Start development live-reload server
PORT=8080 npm run dev

# Start production server:
PORT=8080 npm start
```
